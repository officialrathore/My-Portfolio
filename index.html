HTML

HTML, or Hypertext Markup Language, is the standard markup language for creating web pages and web applications. It defines the structure and layout of a web document by using a variety of tags and attributes

CSS

CSS, which stands for Cascading Style Sheets, is a language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG or XHTML). In simpler terms, CSS is used for styling web pages and controlling their appearance.

Tailwind CSS

Tailwind CSS is a utility-first CSS framework that allows developers to style applications quickly and efficiently by using pre-built utility classes instead of writing custom CSS for each element. With Tailwind, you can directly apply these small, single-purpose classes in your HTML, such as text-center, bg-blue-500, and p-4, which makes building custom UIs fast and straightforward.

JavaScript

"JS" typically refers to JavaScript, which is a widely used programming language for web development. JavaScript (JS) is known for its versatility as it can be used both on the client-side (in web browsers) and server-side (with Node.js).

React.js

React.js, commonly referred to as React, is a JavaScript library used for building user interfaces (UIs) for web applications. It was developed by Facebook and is maintained by both Facebook and a community of individual developers and companies.

Figma

Figma is a cloud-based design tool primarily used for interface design, prototyping, and collaboration. It's widely used by designers, developers, and teams to create user interfaces (UI), user experiences (UX), and other digital products. Unlike traditional design tools that require software installation, Figma runs in a web browser, making it easy to use on any device with internet access. Figma also offers desktop apps for more flexibility.
